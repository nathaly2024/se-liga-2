Explicação sobre o Projeto

Este projeto consiste em criar um site simples usando HTML e CSS para exibir informações sobre um aluno, seu professor, série/turma e a data de entrega de uma atividade. O código foi estruturado para ser claro, modular e fácil de entender. A página é dividida em cabeçalho, corpo principal e rodapé, sendo que cada sessão tem um propósito específico.

No HTML, utilizamos tags como <header>, <main> e <footer> para organizar o conteúdo de maneira semântica. Dentro do <header>, colocamos o nome da escola, enquanto no <main> são apresentadas as informações do aluno. No <footer>, incluímos um aviso de direitos autorais.

No CSS, foram aplicadas regras para tornar o site mais visualmente agradável. Alteramos cores de fundo, fontes e espaçamentos, usando propriedades como background-color, color, padding e margin. Além disso, usamos a técnica de hover para criar um efeito interativo nos botões.
